# Plugins do servidor

Coloque aqui os plugins extras que quiser instalar. Os plugins gerados pelo projeto (`Coins` e `Tags`) ja sao copiados automaticamente para esta pasta ao rodar `./build.sh`.

Os seguintes plugins sao baixados automaticamente pelo container via `SPIGET_RESOURCES`:

- **ViaVersion** (19254) - permite clientes em versoes superiores se conectarem
- **ViaBackwards** (27448) - permite clientes em versoes inferiores se conectarem
- **LuckPerms** (28140) - gerenciador de permissoes
- **Vault** (34315) - API de economia, chat e permissoes
