# Plugins do servidor

Coloque aqui os plugins extras que quiser instalar. Os plugins gerados pelo projeto (`Coins` e `Tags`) ja sao copiados automaticamente para esta pasta ao rodar `./build.sh`.

Os seguintes plugins externos sao baixados automaticamente pelo `../download-plugins.sh` (chamado pelo `start.sh`):

- **ViaVersion** - permite clientes em versoes superiores se conectarem
- **ViaBackwards** - permite clientes em versoes inferiores se conectarem
- **LuckPerms** - gerenciador de permissoes
- **Vault** - API de economia, chat e permissoes
